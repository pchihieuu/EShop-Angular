# How run this project?
## Clone this project
```
git clone https://github.com/pchihieuu/EShop-Angular.git
```
### Install package
```
npm install
npm i -g json-server
npm i fortawesome
npm i swiper
```
### Run project
```
ng serve or npm start
npx json-server --watch db.json
```
### Open testing
```
localhost:4200
localhost:3000
```
